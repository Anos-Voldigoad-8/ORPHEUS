// DEPLOYMENT PROTOCOL

Online in Four
Commands.

01
Clone & Install
Clone the repo and install Python dependencies. Python 3.11+ required. PyAudio and Coqui TTS need system libraries (PortAudio, C++ build tools) on some platforms.
02
Configure Environment
Copy .env.template to .env. Set your LLM provider (Gemini / OpenAI / Ollama), generate your ORPHEUS_MASTER_KEY, and supply any optional API keys for web search.
03
Voice Enrollment
Run intial_setup.py to enroll your voice biometric profile and initialize the encrypted SQLite database. One-time setup — creates your Commander identity.
04
Launch ORPHEUS
Start main.py. FastAPI binds to port 8000. Your browser opens automatically. All five agents come online. The dashboard goes live. ORPHEUS is awake.
bash — deployment sequence
# 01 — Clone & Install
$git clone https://github.com/YOUR/ORPHEUS.git
$cd ORPHEUS && pip install -r requirements.txt

# 02 — Configure environment
$cp .env.template .env
$nano .env # set keys, LLM provider

# 03 — Enroll voice biometrics
$python intial_setup.py
[SUCCESS] User 'Commander' enrolled.
[INFO] Voice Profile Hash: SHA256:a1b2c3...

# 04 — Launch ORPHEUS
